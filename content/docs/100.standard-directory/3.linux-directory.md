### Linux 目录结构

```bash
|- /opt # 第三方软件安装包
  |- java # java 安装包
  |- ...
|- /usr # 用户软件
  |- local # 本地软件
    |- java # java软件
|- /home # 用户目录
|- /bin # 常用指令
|- /boot # 启动文件
|- /dev # 设备文件
|- /etc # 配置文件
|- /lib # 库文件
|- /media # 可移动设备
|- /mnt # 挂载目录
|- /proc # 进程信息
|- /root # root 用户目录
|- /run # 运行时信息
|- /sbin # 管理员常用指令
|- /srv # 服务数据
|- /sys # 系统信息
|- /tmp # 临时文件
|- /var # 日志
|- /lost # 文件系统恢复
```
