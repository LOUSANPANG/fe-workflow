## SEO

- SSR 服务端渲染
- HTML 文件
  - Title(标题)、Description（描述）和 Keywords（关键词）
  - Meta 标签使用
  - Open Graph 协议
  - 结构化数据，例如 JSON-LD
- 代码方面
  - HTML 语义化
  - a 标签中 rel 属性，nofollow、external
- 额外文件
  - 添加 sitemap.xml 站点地图
  - 添加 robots.txt 文件
- 各搜索引擎提交站点收录
