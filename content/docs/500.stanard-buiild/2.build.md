## 资源优化

### 静态资源
- 图片压缩
- HTML压缩
- 配置 externals 引入 cdn 资源

### js
- 压缩 js 文件
- 拆包 & 对分割的代码 chunk 进行预加载

### css
- 抽离和压缩样式文件
- 去除多余无效样式文件

### 系统
- 开启GZIP压缩
- Tree Shaking
- 剔除日志

### 优化查询
- resolve.alias 减少查找过程
- resolve.extensions 减少查找过程
- noParse 配置不解析的模块
- IgnorePlugin 忽略部分模块
- 缩小 loader 搜索范围
